About

Regular Expression creator is a pet project of mine. The goal of the project has so far been to create an offline version of Regexr for my personal use with the possibility of more features (such as visualizations like Regulex, support for multiple flavors, etc.). The current status of the project is barely alpha ;). All it does right now is highlight text in the output box that matches the expression in the input box, editing the contents of the output box is very glitchy, and there are probable as-of-yet undiscovered fatal bugs. I really don't know why I'm writing this, since I probably won't release this anytime soon, but oh well... This project will probably be released on Github eventually, and my personal website javascriptftw.github.io (it needs some stuff on it ;) ).
